## 1. 起点中文网

### 1.1 平台概况

| 项目 | 信息 |
|------|------|
| 所属公司 | 阅文集团（腾讯控股） |
| 创立时间 | 2003年 |
| 定位 | 中国网络文学行业龙头，以男频付费阅读为核心，同时运营女频品牌 |
| 用户量级 | 注册用户超过数亿，阅文集团整体MAU约3亿+ |
| 官网 | https://www.qidian.com |

### 1.2 流量情况

- 阅文集团旗下产品（含起点读书APP、QQ阅读等）合计DAU数千万级别
- 起点中文网PC端+起点读书APP是男频网文最大流量入口
- 腾讯生态导流优势显著（微信读书、QQ浏览器等渠道）

### 1.3 新手作者扶持政策

- **新人训练营**：提供写作指导课程、编辑一对一交流机会
- **新书期推荐**：新书上架后有一定的免费推荐位展示期
- **签约即有保底**：部分合同提供基础保底稿酬（具体金额视合同等级而定）
- **新人月票奖**：新书期间月票排行有额外奖金
- **完本奖励**：完结作品可获得额外奖金
- 注意：起点的扶持力度与作品数据强相关，头部效应明显

### 1.4 商业模式

- **核心模式**：VIP付费阅读（按章节/千字收费）
- **免费模式**：起点也推出了免费阅读频道（广告变现）
- **打赏系统**：读者可对作品打赏（月票、推荐票、赏银等）
- **IP开发**：优秀作品可获得影视、动漫、游戏改编机会
- **粉丝经济**：盟主制度、粉丝等级体系完善

### 1.5 稿酬体系

| 项目 | 标准 |
|------|------|
| 订阅分成 | 作者可获得订阅收入的约50%（净收入分成） |
| 千字单价 | 普通签约：订阅量 × 千字价格（5分/千字起），头部作品可达数元/千字 |
| 全勤奖 | 普通签约：每月约600-1000元（需每日更新4000字以上） |
| 打赏分成 | 作者获得约50% |
| 保底稿酬 | 根据合同等级不同，从数百元到数万元/月不等 |
| 完本奖 | 依据总字数和完结质量，数千到数万元 |

**注意**：2020年后阅文集团合同改革，新合同条款涉及著作权归属争议较大，作者需仔细阅读合同条款，特别关注版权归属和改编权条款。

### 1.6 发布门槛

- **注册要求**：实名认证（身份证+银行卡）
- **签约门槛**：一般需发布3-5万字以上，由责任编辑审核决定是否签约
- **审核机制**：有专职编辑团队，新书审核通常1-3个工作日
- **签约合同**：分为普通签约、精品签约、大神签约等不同级别
- **竞争强度**：极高，每日新书数量庞大，脱颖而出难度大

### 1.7 适合的小说类型

- **强势类型**：玄幻、仙侠、都市、科幻、游戏、历史（男频为主）
- 起点女生网（现已整合）覆盖言情、古言等女频类型
- 短篇不是起点的主要方向，以长篇连载为核心（百万字级）

### 1.8 API/自动化发布可行性

- **官方API**：起点/阅文作家后台无公开的写作发布API
- **作家助手APP**：阅文集团提供"作家助手"APP进行移动端写作和发布
- **自动化可行性**：较低。需通过作家助手或网页后台手动操作，存在登录验证和反爬机制
- **第三方工具**：市面上有一些非官方的批量发布工具，但使用风险较高，可能违反平台规定

## 2. 番茄小说

### 2.1 平台概况

| 项目 | 信息 |
|------|------|
| 所属公司 | 字节跳动（抖音集团） |
| 创立时间 | 2019年 |
| 定位 | 免费网络小说平台，以广告变现为核心模式 |
| 用户量级 | DAU超过1亿（截至2024-2025年数据），增长极为迅速 |
| 官网 | https://fanqienovel.com |

### 2.2 流量情况

- DAU超过1亿，是目前国内流量最大的网文阅读平台之一
- MAU超过3-4亿
- 依托字节跳动的今日头条、抖音等产品导流，增长势头强劲
- 下沉市场用户占比较高

### 2.3 新手作者扶持政策

- **零门槛发布**：注册即可发布，无需先签约
- **新人激励计划**：新作者前期有流量扶持和现金激励
- **创作激励计划**：根据阅读量给予稿酬（读完率越高收益越高）
- **签约制度**：达到一定字数和数据后可签约，签约后收益更高
- **征文活动**：定期举办征文比赛，奖金丰厚（万元至十万元级别）
- **短篇扶持**：对短篇小说（几万字级别）也有专项扶持

### 2.4 商业模式

- **核心模式**：免费阅读 + 广告变现（信息流广告、激励视频广告等）
- **付费模式**：部分作品也支持付费阅读
- **创作激励**：平台按阅读量向作者分配广告收益
- **IP改编**：与字节跳动旗下影视业务联动

### 2.5 稿酬体系

| 项目 | 标准 |
|------|------|
| 收益模式 | 按有效阅读量计算广告分成收入 |
| 千字收益 | 非签约：约1-5元/万次有效阅读；签约后收益倍率更高 |
| 全勤奖 | 有全勤奖制度，约200-600元/月（需每日更新） |
| 签约保底 | 部分签约合同有保底，金额视作品质量而定 |
| 完本奖 | 有完本奖励，鼓励作者完结作品 |
| 爆款潜力 | 因流量大，爆款作品月收入可达数万甚至数十万元 |

**核心特点**：收入与阅读量（尤其是读完率）直接挂钩，流量大但单次阅读收益较低，靠量取胜。

### 2.6 发布门槛

- **注册要求**：手机号注册+实名认证
- **发布门槛**：极低，注册后即可直接发布作品
- **签约条件**：一般需要连载2-3万字以上，数据达到一定标准后编辑主动联系或作者申请签约
- **审核机制**：机器审核为主，速度快，通常几小时内通过
- **竞争环境**：作品数量极多，但流量也大，新人有机会获得推荐

### 2.7 适合的小说类型

- **强势类型**：现代言情、都市、甜宠、悬疑、赘婿、逆袭类（偏大众、快节奏）
- 男频女频均有大量读者
- **短篇友好**：平台对中短篇小说（5-30万字）也有较好的生态
- 偏向通俗易读的风格，纯文学类不太适合

### 2.8 API/自动化发布可行性

- **官方API**：番茄小说作者后台（创作者中心）无公开API
- **网页端操作**：可通过 https://writer.fanqienovel.com 进行发布
- **自动化可行性**：中等。网页后台结构相对简单，技术上可以通过模拟浏览器操作实现自动化，但存在风控风险
- **移动端**：提供番茄作者APP

## 3. 晋江文学城

### 3.1 平台概况

| 项目 | 信息 |
|------|------|
| 所属公司 | 北京晋江原创网络科技有限公司（独立运营） |
| 创立时间 | 2003年 |
| 定位 | 国内最大的女频原创文学网站，耽美/言情重镇 |
| 用户量级 | 注册用户数千万，核心活跃用户以女性为主 |
| 官网 | https://www.jjwxc.net |

### 3.2 流量情况

- DAU约数百万级别（PC+APP合计）
- 用户粘性极高，付费意愿强
- 在女频市场占据统治地位
- 用户以18-35岁女性为主，学历和消费能力偏高

### 3.3 新手作者扶持政策

- **入驻门槛低**：注册即可发表，无需先签约
- **签约即入V**：作品达到一定字数可申请入V（VIP收费章节）
- **新人榜单**：设有新人榜、新书榜等推荐位
- **编辑推荐**：好的作品会获得编辑主动推荐
- **晋江特色**：社区氛围好，读者互动活跃，口碑传播效应强
- **扶持力度**：相对起点而言，晋江对新人的主动扶持较少，更依赖作品自身质量和口碑

### 3.4 商业模式

- **核心模式**：VIP付费阅读（按章节收费，千字3分晋江币起）
- **打赏系统**：读者可打赏（霸王票等）
- **包月制**：提供包月VIP服务
- **IP改编**：晋江是影视改编的重要作品来源地（如《花千骨》《香蜜沉沉烬如霜》等均出自晋江）
- **实体出版**：优秀作品有出版机会

### 3.5 稿酬体系

| 项目 | 标准 |
|------|------|
| 订阅分成 | 作者获得订阅净收入的约50%（基础分成），优秀作品可更高 |
| 入V条件 | 一般需要连载满一定字数（通常6万字以上）且有一定收藏量 |
| 全勤奖 | 晋江无传统全勤奖制度 |
| 打赏分成 | 作者获约50% |
| 积分制度 | 晋江独特的积分排行影响推荐曝光 |
| 收入特点 | 头部作品收入极高，腰部作品也有稳定收入，但尾部作品收入较低 |

### 3.6 发布门槛

- **注册要求**：邮箱注册，发布需实名认证
- **发布门槛**：极低，注册后即可发表
- **签约/入V**：需达到字数要求和一定的数据表现
- **审核机制**：有人工审核，对内容合规性要求较严（尤其是涉及敏感内容）
- **特殊要求**：晋江对防盗版措施较严，对作品首发独家有要求

### 3.7 适合的小说类型

- **强势类型**：言情（古代/现代/幻想）、耽美（BL）、百合（GL）、纯爱、宫斗、宅斗
- 女频绝对主场，男频作品较少
- 文笔要求相对较高，读者群体对文学性有一定追求
- 短篇、中篇也有生存空间（晋江有短篇频道）

### 3.8 API/自动化发布可行性

- **官方API**：无公开API
- **网页端**：通过晋江作者后台发布，界面较为传统
- **自动化可行性**：低至中等。晋江网站技术架构较老，但有反爬和验证码机制
- **注意**：晋江对自动化行为比较敏感，不建议使用自动化工具

## 4. 红袖添香

### 4.1 平台概况

| 项目 | 信息 |
|------|------|
| 所属公司 | 阅文集团（腾讯控股） |
| 创立时间 | 2001年（后被阅文集团收购） |
| 定位 | 女频网络文学平台 |
| 用户量级 | 注册用户千万级，但活跃度近年有所下降 |
| 官网 | https://www.hongxiu.com |

### 4.2 流量情况

- 并入阅文体系后，独立流量有所萎缩
- DAU约百万级别
- 部分流量已导向阅文旗下其他产品
- 品牌影响力仍在，但增长乏力

### 4.3 新手作者扶持政策

- 与阅文集团统一的签约体系
- 新人福利与起点体系类似但规模较小
- 编辑团队相对较小
- 作为阅文旗下品牌，共享阅文的作者福利体系

### 4.4 商业模式

- 与阅文集团整体模式一致
- VIP付费阅读为主
- 共享阅文的分发渠道

### 4.5 稿酬体系

| 项目 | 标准 |
|------|------|
| 订阅分成 | 与阅文体系一致，约50%净收入分成 |
| 全勤奖 | 有，标准与阅文体系一致 |
| 整体收益 | 因流量不如起点，同等质量作品收入通常低于起点 |

### 4.6 发布门槛

- 实名认证
- 签约门槛与阅文体系一致
- 竞争压力相对起点较小

### 4.7 适合的小说类型

- **强势类型**：现代言情、古代言情、总裁文、甜宠文
- 纯女频定位
- 偏向通俗言情风格

### 4.8 API/自动化发布可行性

- 使用阅文集团的作家助手系统
- 自动化可行性与起点类似，较低

## 5. 七猫小说

### 5.1 平台概况

| 项目 | 信息 |
|------|------|
| 所属公司 | 上海七猫文化传媒有限公司（百度投资） |
| 创立时间 | 2018年 |
| 定位 | 免费网络小说平台，与番茄小说直接竞争 |
| 用户量级 | DAU数千万级别 |
| 官网 | https://www.qimao.com |

### 5.2 流量情况

- DAU约3000-5000万（2024-2025年数据）
- MAU超过1-2亿
- 在免费阅读赛道排名前三
- 百度系流量支持（百度搜索、百度APP等导流）

### 5.3 新手作者扶持政策

- **零门槛入驻**：注册即可发布
- **新人激励**：有新人专属激励计划，按阅读量给予收益
- **流量扶持**：新书有一定的推荐曝光机会
- **征文活动**：定期举办征文活动，奖金可观
- **签约激励**：签约后收益倍率提升

### 5.4 商业模式

- **核心模式**：免费阅读 + 广告变现（与番茄小说模式相似）
- **付费内容**：部分精品内容支持付费
- **创作激励**：按阅读量分配广告收益给作者

### 5.5 稿酬体系

| 项目 | 标准 |
|------|------|
| 收益模式 | 按有效阅读量计算收益 |
| 全勤奖 | 有，约200-500元/月 |
| 签约保底 | 部分签约合同有保底 |
| 收入水平 | 与番茄小说相近，但因流量略低，整体收入略低 |

### 5.6 发布门槛

- 手机号注册+实名认证
- 发布门槛极低
- 审核以机器审核为主，速度快
- 签约需达到一定字数和数据

### 5.7 适合的小说类型

- **强势类型**：都市、言情、玄幻、悬疑、赘婿、甜宠
- 男频女频均有覆盖
- 偏向通俗、快节奏的大众类型
- 中短篇也有市场

### 5.8 API/自动化发布可行性

- **官方API**：无公开API
- **作者后台**：通过七猫作者中心网页端发布
- **自动化可行性**：中等，与番茄小说类似

## 6. 掌阅（iReader）

### 6.1 平台概况

| 项目 | 信息 |
|------|------|
| 所属公司 | 掌阅科技股份有限公司（A股上市公司） |
| 创立时间 | 2008年 |
| 定位 | 综合性数字阅读平台，兼顾出版物和网络文学 |
| 用户量级 | 累计注册用户超过7亿，MAU约1-2亿 |
| 官网 | https://www.zhangyue.com |

### 6.2 流量情况

- DAU约2000-3000万
- 在综合阅读APP中排名前列
- 用户付费习惯较好
- 覆盖面广，包括出版物、网文、漫画等多品类

### 6.3 新手作者扶持政策

- **掌阅文学**：旗下原创文学品牌，接受投稿
- **新人签约**：审核相对严格，但签约条件较为优厚
- **编辑对接**：签约后有专属编辑指导
- **推荐资源**：签约作品可获得掌阅APP内推荐位
- **出版机会**：优秀作品有出版和IP改编机会

### 6.4 商业模式

- **核心模式**：付费阅读（按本/按章）
- **包月会员**：VIP会员可免费阅读部分作品
- **广告模式**：部分免费作品通过广告变现
- **硬件生态**：掌阅iReader电子书阅读器

### 6.5 稿酬体系

| 项目 | 标准 |
|------|------|
| 订阅分成 | 约50%净收入分成（签约作品） |
| 买断模式 | 部分作品可选择买断，千字30-200元不等 |
| 全勤奖 | 有，标准与合同约定为准 |
| 保底稿酬 | 签约作品通常有保底 |
| 收入特点 | 付费用户质量高，签约作品收入相对稳定 |

### 6.6 发布门槛

- **投稿要求**：需提交大纲和样章（通常3万字以上）
- **审核机制**：人工编辑审核，周期较长（1-2周）
- **签约门槛**：相对较高，注重作品质量
- **实名认证**：必须

### 6.7 适合的小说类型

- **强势类型**：现代言情、古代言情、都市、玄幻、悬疑推理
- 男频女频均有
- 对文笔有一定要求
- 中长篇为主

### 6.8 API/自动化发布可行性

- **官方API**：无公开的作者发布API
- **作者后台**：掌阅有作者管理后台
- **自动化可行性**：低。投稿审核流程较为严格，不适合自动化批量发布

## 7. 纵横中文网

### 7.1 平台概况

## 附录：新手作者发布策略建议

### A. 推荐发布策略

1. **首选番茄小说作为主要平台**：零门槛、高流量、收益透明
2. **同步七猫小说**（如无独家协议限制）：增加曝光渠道
3. **女频作者首选晋江文学城**：女频生态最好
4. **作品成熟后考虑起点中文网**：追求更高的天花板和IP开发
5. **多平台试水**：初期可在2-3个平台同时发布，观察数据后集中精力

### B. 关于API和自动化发布

目前各主流平台均未提供官方的写作发布API，自动化发布的可行性总体较低：

| 可行性等级 | 平台 | 说明 |
|------------|------|------|
| 中等 | 番茄小说、七猫小说 | 网页后台结构相对简单，可通过Selenium等工具模拟操作 |
| 低-中 | 晋江、17K、纵横 | 有一定反爬措施，但技术门槛不高 |
| 低 | 起点、掌阅、红袖 | 阅文系统反爬严格，掌阅审核流程不适合自动化 |
| 较高 | 逐浪 | 技术架构老旧，但平台本身意义不大 |

**重要提醒**：
- 自动化发布存在**账号封禁风险**，各平台均在用户协议中禁止使用自动化工具
- 建议使用各平台官方提供的**作家助手APP**或**网页端后台**手动发布
- 如确需多平台同步，可考虑使用**剪贴板工具**提高复制粘贴效率，而非完全自动化
- 部分第三方写作工具（如壹写作、码字精灵等）支持多平台格式导出，可提高效率

### C. 合同签署注意事项

1. **版权归属**：重点关注著作权是否完全转让，还是仅授权
2. **独家条款**：注意是否要求全网独家，是否限制在其他平台发布
3. **改编权**：影视、游戏、动漫改编权的归属和分成比例
4. **合同期限**：注意合同有效期和自动续约条款
5. **解约条件**：了解提前解约的条件和违约金
6. **收益结算**：确认结算周期、最低提现金额等细节
