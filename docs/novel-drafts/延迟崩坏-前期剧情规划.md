# 《延迟崩坏》前期剧情规划（修订版）

## 核心改动

- 母亲之死从第7章推迟到第10章，给末日初期的母子+许昭+温以宁共处留出2-3章空间
- 崩坏从单章扩展为三章（6校园线→7双线并行→8重逢），后续章节顺延两章
- 新增温以宁（舞蹈特长生，侧分披肩黑发，温柔型）作为第四核心成员，校园线搭档→小队协调/后勤
- 赵磊的复杂背景线暂时搁置，前期只作为普通霸凌者存在，中后期再展开
- **小周是母亲之死的真正凶手**——表面是"后门被丧尸撞开"的意外，实际是小周蓄意松动门挡。许昭以为是自己没守住门，背负虚假的罪疚感
- **西装男是红鲱鱼**——刁难沈默、盯温以宁、偷吃物资，转移读者对真凶的注意力。ch10转移前主动离队退场
- 母亲之死的情感重量从"铺垫→失去"升级为"铺垫→重逢→共处→失去"，同时增加了真相揭露时的二次冲击

## 串联事件

**事件一：秋季体测**（贯穿2-4章，已完成）
- 系统探索与身体蜕变的量化展示场景
- 许昭登场、苏檀登场
- 赵磊推不动的冷处理反转

**事件二：城市异常**（第4章末，已完成）
- 99%→99.09%，末日倒计时推进
- 母亲凌晨两点还在卸货，沈默发微信确认安全

**事件三：丧尸爆发**（第6-8章）
- 99.1%之后突然爆发，全城陷入混乱
- 第6章：校园线——沈默+温以宁困守舞蹈教室后骑摩托逃离
- 第7章：双线并行——舞蹈教室脱出 + 许昭在便利店与林岚汇合
- 第8章：沈默到达便利店，母子重逢，处理便利店人性冲突，C警告48h内更严重的第二波

**事件四：母亲之死**（第10章）
- **表面**：转移途中居民楼后门被丧尸攻破，许昭冲出去堵门，母亲挡在温以宁前面被咬
- **真相**：小周（便利店店员）蓄意松动后门门挡，制造"意外"。动机是嫉妒沈默拥有的一切（能力、母亲的爱、温以宁的好感）
- **红鲱鱼**：西装男在ch8-9持续作妖（刁难、偷吃、盯温以宁），ch10出发前主动离队——读者以为他后面会回来搞事
- 四条隐藏线索埋入ch10，揭露时才被读者回溯发现

## 章节展开

### 第5章：暴风雨前

**末日预兆收紧：**
- 网上帖子：某市集体昏厥、有人拍到天空半透明建筑轮廓（秒删）
- 沈默联想系统和城市异常，笔记本写下："如果到达100%，会发生什么？"
- 身体接近蜕变完成态，系统参数树基本摸清

**关系线收束：**
- 赵磊不再直接动手，改让跟班找茬、背后说坏话。压迫感从肉体转向社交孤立
- 放学后跟班堵沈默，不是动手，而是社交羞辱——当着人翻他书包、把笔记本扔来扔去。沈默有实力硬来但选择不动手（动手只会暴露更多）
- **许昭介入**：路过看见，二话不说冲进来，一把搂住沈默肩膀，大大咧咧喊"教练叫你去田径队试训，走走走"，硬把人从包围圈里拽走。跟班不敢惹高三体育生，只能看着
- 事后沈默说"你不用管我，我能处理"。许昭："我知道你能，但我看见了就忍不住。"——**这句话是第10章的种子：在学校里这是仗义，在末日里同样的冲动被小周利用**
- **沈默的不适感**：回家路上，心里不是感激而是微妙的不适。他习惯独自消化所有问题，别人的介入让他觉得控制权被侵犯了——哪怕是好意。但他无法否认许昭出现那一刻胸口确实松了一口气。这种矛盾让他烦躁。笔记本上写：【许昭——不可控变量。但目前收益为正。】写完又觉得用这种方式定义一个帮过自己的人有些刻薄，但他不知道别的方式
- 这段情绪接上母亲线：回到家情绪有点乱，母亲看出来但不问，饭桌上多夹了一次菜。沈默的"妈，你别太累了"不只是关心，还带着被善意击中后的一点柔软
- 苏檀第一次主动问沈默一道物理题。沈默回答了，苏檀点点头走了。第一次平视

**母亲——最后的平静（最重要的一场戏）：**
- 周末晚上母亲难得休息。两人吃简单晚饭——白粥、咸菜、一盘炒青菜
- 沈默说"妈，你最近别太累了"。大概是他这一个月说过的最长的关心话
- 母亲愣了一下，笑笑说"知道了"
- 这顿饭就是最后一顿"正常世界"的晚餐。读者后面会回想这个画面

**末尾钩子：**
- 当晚深夜看系统：99.1%
- 合上笔记本，说不清的不安。窗外月亮被一层不属于云层的灰白色遮住一瞬。眨眼恢复正常

### 第6章：崩坏·校园线（9月15日·周一）

**爆发：**
- 99.1%之后突然爆发，天空浑浊、远处建筑传来碎裂声
- 校园内外同时出现第一批丧尸（和菜市场老周一样的浑浊感）
- 系统界面出现大量新信息，沈默来不及细看

**校园逃离：**
- 沈默冷静分析丧尸特征：趋声、<5km/h、不可逆
- 冲向高三(1)班找许昭，途中格斗术实战首秀，多次战斗（1/10→4/10）
- **关键发现**：实战提升远超训练，格斗术升级反向拉动基础属性突破5.5瓶颈
- 苏檀被咬——"走。"（她最后的台词）；赵磊被丧尸拉走

**温以宁登场（高三1班教室内）：**
- 许昭已经不在，教室角落蹲着温以宁——侧分披肩黑发，穿真理裤，刚练完早功课没换回校服
- 沈默从窗口观察：校门口聚集大量丧尸，无法从正门离开
- 温以宁提出：舞蹈教室在艺术楼三楼，偏僻、好守，"钥匙还没还"
- 两人穿连廊到艺术楼，温以宁开门反锁，器材柜+钢琴堵门
- 给母亲发消息：已读，未回复

### 第7章：困兽（双线并行）

**A线·舞蹈教室→脱出：**
- 盘点物资（钢把杆、急救箱、粉笔、半桶水）
- 温以宁人物展开：军人父亲已故，母亲改嫁，外婆去世后独居
- 观察丧尸规律：白天迟钝，游荡状态<2km/h，主要依赖听觉
- 下午丧尸被远处车声吸走→翻窗→路上遇丧尸（格斗术4/10→5/10）→教职工停车棚→摩托
- 北门外清理丧尸群（格斗术5/10→7/10），温以宁见识到沈默的战斗力
- 方向：母亲工作的便利店

**B线·许昭+便利店（时间回溯到爆发当天上午）：**
- 许昭翻窗逃出学校，记得沈默说过母亲在东区便利店
- 便利店内幸存者群像：
  - 林岚（沈默母亲）：已在整理物资、堵后门
  - 小周（店员）：吓得发抖但听指令
  - 刘姓西装男：资源争夺者
  - 年轻母亲+婴儿：婴儿哭声吸引丧尸
  - 大学生男：暗藏咬伤，逐渐恶化
- 许昭和林岚发现大学生被咬，但未声张
- 章末：远处传来摩托引擎声——沈默到了

### 第8章：重逢

**两线交汇：**
- 摩托穿越城区，路上多次清理丧尸，格斗术突破初级→中级
- 到达时：格斗术中级3/30（业余拳击手水平），力量/耐力/敏捷均6.0，视觉5.3，身高177
- 母子重逢：林岚没哭，摸了摸他的脸确认是真的。注意到儿子又长高了、皮肤变好了，没问

**暗线启动——双重视线：**
- 温以宁进入便利店时所有人都看了她一眼。刘姓西装男目光从脸到腿扫了一遍（明目张胆）；小周只看了一眼就低头了，耳朵尖红了（"害羞"）
- 许昭打趣沈默"把班花都拐上了"，温以宁脸红说"别乱说"

**西装男红鲱鱼：**
- 西装男反对收人，挡在过道里刁难沈默。许昭往前一站185的块头压住他
- 大学生咬伤曝光，西装男嚷着赶人。林岚端热水过去："走不走的等天亮再说"

**小周暗种：**
- 温以宁帮沈默换纱布→小周端水来搭讪温以宁→"谢谢"后转头继续帮沈默→小周愣住退回（搭讪失败，看起来只是腼腆）
- 值班分配：小周主动请缨值后门最后一班。林岚夸"关键时候靠得住"
- 深夜：小周独自坐在后门旁，安静盯着门锁看——表情在昏暗灯光下看不清

**C的警告：**
- 信号短暂恢复，C发来消息：全球多城同时出事，48h内第二波更严重，不要留在建筑密集区
- 沈默看完翻过手机——没有告诉任何人

### 第9章：秩序

**便利店第2-3天——短暂的"家"：**
- 沈默接管秩序：物资清点（水3天/食物5天/药品有限）、白天侦察、夜间封锁
- 大学生恶化后主动离开（"趁我还能走"），沈默没拦
- 四人小队配合默契：沈默（分析+决策）、许昭（体力+侦察）、温以宁（协调+照顾）、母亲（后勤+秩序+情绪稳定）

**西装男红鲱鱼加深：**
- 大学生走后西装男觉得自己"主张正确"更嚣张
- 被林岚发现偷藏方便面和水，林岚没当众揭发只是少分他一包
- 看林岚的眼神变阴冷——读者被引导觉得"这家伙迟早搞事"

**温以宁与沈默关系推进：**
- 出门前温以宁帮沈默系护腿板绑带，"小心点"，沈默出门前回头看了她一眼
- 晚上靠着同一排货架，温以宁问他在看什么，靠得更近了一点

**小周暗线加深：**
- 小周看到温以宁帮沈默系装备，码货的手停了一秒
- 温以宁和沈默靠在一起时，小周拿着水想走过去，看到那个画面后默默退回后门
- 视线在温以宁和沈默之间来回，最后落在自己手上——手指在裤腿上无意识摩擦
- **关键伪装**：小周夜间又坐后门值班，用手电照门缝——实际不是看外面，是在看门锁结构，手指描摹锁扣形状。没人注意

**母亲与许昭的温情：**
- 许昭狼吞虎咽，母亲说"慢点，又没人跟你抢"——许昭嘴角咧了一下（没人这样嘱咐过他）
- 许昭对沈默说"温以宁对你挺好的"→沈默不接话→许昭："行，装。"

**转移决定：**
- 第三天沈默把C的48h警告告诉许昭，决定今天转移
- 母亲听完没问为什么："我去收拾东西。"按人头分装物资——包括西装男那份

### 第10章：代价

**母亲之死。表面看是转移途中避难点被丧尸攻破的意外。真相是小周蓄意制造。**

设计原则：①许昭不是凶手但以为自己是 ②小周的行为在混乱中被完美掩盖 ③读者第一遍以为是意外+许昭冲动 ④四条线索埋入供揭露时回溯

**出发——西装男退场（红鲱鱼离场）：**
- 母亲在门上贴纸条："里面有水和饼干，自取"
- 西装男选择单独行动，拿了超额物资走了。许昭想拦，沈默摇头
- **西装男退场——读者以为他后面还会回来搞事，实际不会再出现**

**转移途中——许昭冲动伏笔：**
- 城市比两天前更糟，灰白雾气、温度异常区、更多裂缝
- 路上遇到被困幸存者，许昭不顾劝阻冲去救人→暴露位置引来丧尸→沈默断后
- 沈默："下次先跟我说。"许昭："我控制不住。"
- 母亲晚上给沈默端热水："你那个朋友，心眼好，就是太冲了。你看着他点。"

**居民楼落脚——小周的最后准备：**
- 三层小楼，前后两门，窗户用家具堵。沈默亲手检查所有入口，安排值班
- 后门又是小周——连续三天值后门，大家习以为常，觉得他"认真负责"

**深夜——小周动手：**
- 所有人睡了，小周把堵后门柜子底部两个支撑脚悄悄掰松——柜子看起来还堵着门，但外力一推就倒
- 回到位置坐下闭眼
- *（回溯视角·揭露时才展开）小周的内心：从便利店第一晚看温以宁帮沈默换纱布起就开始发酵。那个高中生比他小、长得好看、能打、有母亲的爱有女生的温柔。而他22岁，月薪三千五，没人关心。末日前loser，末日后还是。他不想杀谁，只是想看看"完美"碎掉的样子。*

**凌晨三点——丧尸涌入：**
- 丧尸群撞后门，柜子晃两下就倒了
- 小周第一个喊"后门破了！！"——时机恰到好处（不太早也不太晚）
- 许昭从二楼跳下冲向后门堵丧尸，185的块头在窄门口挡住一波
- 外面传来幸存者求救声（白天路上救过的人），许昭被迫退到院子里

**母亲之死：**
- 林岚在二楼下楼找沈默，楼梯口侧窗翻进一个丧尸。温以宁尖叫
- 林岚挡在温以宁前面
- 沈默从一楼往上冲，隔了半层楼
- 赶到时母亲倒在楼梯转角，颈侧有咬痕。她看着儿子，说"没事的"——到死都在说没事的
- 沈默握着她的手，看着她眼睛一点一点失去光。没哭，没喊，没动

**许昭崩溃（虚假的罪疚）：**
- 许昭退回来看到这一幕，浑身是血，以为是自己没守住后门
- 嘴唇动了动说不出话。沈默没看他
- 沈默把母亲放平，脱校服盖上，站起来说"走"——声音平得像说别人的事

**小周的伪装：**
- 小周搀着年轻母亲出来，表情"恰到好处的惊恐和悲伤"，在发抖——但瞳孔是稳的。没人仔细看

**四条埋藏线索（读者第一遍不会注意）：**
1. 柜子倒得太轻易——沈默亲手堵的，不应该几个丧尸就推倒
2. 小周喊的时机——一直醒着盯门，为什么柜子晃动时没去顶住？
3. 小周在混乱中的位置——没冲向后门帮忙，第一时间跑到二楼"疏散"（实际远离危险）
4. "害怕到发抖"的人怎么还有余力搀别人？

## 母亲之死的设计理由

### 三层因果——表面/误解/真相

**表面（所有角色看到的）：** 后门被丧尸撞开→许昭冲出去堵门→母亲下楼被丧尸咬。一场"意外"。

**许昭的误解：** 他以为是自己没守住门、冲出去救人导致防线崩溃。性格铺垫完美吻合（ch5"看见了就忍不住"→ch10冲出去堵门），读者第一遍也这样理解。

**真相（后续揭露）：** 小周蓄意松动门挡。许昭的冲动不是原因，是被利用的工具——小周算准了许昭会往外冲，这样混乱更大、更没人注意后门的问题。

### 多重矛盾（升级版）
- 沈默恨许昭吗？→ 许昭不是凶手，但他自己不知道
- 许昭的罪疚是虚假的吗？→ 不是。他确实冲出去了，确实没守住。只是根本原因不在他
- 揭露真相后，许昭是解脱还是更痛苦？→ 更痛苦——原来他背负的不是自己的罪，是被人设计的
- 沈默恨小周吗？→ 小周不是恶魔，只是一个在末日前就被世界抛弃的loser。恨他太容易了，也太廉价了
- **核心问题**：善良的人为什么总是死？是因为身边有恶意的人，还是因为善良本身就是弱点？

### 许昭的救赎弧线（两段式）
**第一段（揭露前）：虚假的赎罪**
- 从"热情开朗"变成沉默寡言，每次行动前看沈默等他点头
- 不敢道歉，用行动赎罪——走最前面、挡在沈默前面、执行最危险任务
- 他和沈默之间的沉默本身就是一种刑罚

**第二段（揭露后）：真正的愤怒**
- 得知真相后，许昭的反应不是释然而是暴怒——他背了这么久的债，竟然是被人设计的
- 对小周的愤怒，对自己"被利用了还不知道"的屈辱感
- 但随后更深的痛苦：即使门挡没被动过，如果他没冲出去，母亲可能不会死。小周利用的正是他最真实的性格

**最终：** 后期某个关键时刻，许昭为救沈默而死。死前可能说："你妈说过，让我慢点吃，又没人跟我抢。"——把读者拉回第8章便利店里的那个夜晚

### 小周作为反派的设计优势
1. **不需要"大恶人"**：小周不是邪恶天才，只是一个被嫉妒吞噬的普通人。这比"大BOSS害死母亲"更现实、更恐怖
2. **伏笔自然**：他在ch8-9的行为（值夜班、看门锁、搭讪失败）都可以解读为"老实人在努力融入"，揭露后才变成犯罪证据
3. **红鲱鱼有效**：西装男吸走了所有"坏人"注意力，小周的"好人"伪装完美
4. **二次冲击**：读者第一遍以为理解了母亲的死（许昭冲动），揭露时发现完全不是——这种叙事反转比一次性的悲剧更有力

## 赵磊的处理

前期（1-11章）赵磊只作为校园霸凌者存在，体测后淡出。末日爆发后暂不出现。

中后期再展开赵磊线：
- 再次相遇时赵磊已经变了（末日改变了所有人）
- 他的原生家庭背景（父亲酗酒家暴、母亲跑了）可以在那时候再揭示
- 赵磊可以走一条独立的成长线，不需要跟母亲的死绑定

## 前四章已写内容的修正清单

以下问题需要在修改现有章节时处理：

1. **删除装备检测**：第2章末尾的"检测到装备更新：基础运动鞋（白）"和敏捷修正，与世界观矛盾，直接删掉。跑鞋场景保留，去掉系统反应即可
2. **压缩身高增速**：168→172（前4章内），175放到第5-6章。4cm在青春期可解释，7cm太扎眼
3. **C加灰度**：在现有章节中找1-2处让C的回复有微妙的回避或延迟，暗示她有自己的agenda。不需要大改，细节调整即可
4. **许昭人设定稿**：以实际写出的热情型为准（高三体育特长生、自来熟、聒噪但不令人厌恶），前期规划中"安静型"的描述作废
5. **赵磊铺垫不急**：既然不再需要第5章就用赵磊的家庭线，前4章的赵磊保持扁平霸凌者即可，不需要补淤青等细节
